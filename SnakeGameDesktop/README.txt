Klasik Yılan Oyunu - C# WinForms

Çalıştırma:
1) Bilgisayarında .NET 8 SDK kurulu olmalı.
2) Terminali bu klasörde aç.
3) Şu komutu çalıştır:
   dotnet run

EXE almak için:
   dotnet publish -c Release -r win-x64 --self-contained true

Yayın sonrası exe dosyası şu klasörde oluşur:
   bin\Release\net8.0-windows\win-x64\publish\

Kontroller:
- Hareket: Yön tuşları veya WASD
- Yeniden başlat: Enter
- Çıkış: Esc
