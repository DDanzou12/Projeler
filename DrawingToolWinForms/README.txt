DrawingToolWinForms
===================

İçerik:
- DrawingToolWinForms.slnx  -> Yeni XML solution formatı
- DrawingToolWinForms.sln   -> Geriye uyumluluk için klasik solution dosyası
- DrawingToolWinForms/      -> Windows Forms .NET Framework 4.8 projesi

Özellikler:
- Kalem
- Çizgi
- Dikdörtgen
- Elips
- Silgi
- Renk seçme
- Fırça kalınlığı ayarlama
- Tuvali temizleme
- PNG olarak kaydetme

Açma:
1. Visual Studio ile DrawingToolWinForms.slnx dosyasını aç.
2. Eğer Visual Studio sürümün .slnx desteklemiyorsa DrawingToolWinForms.sln dosyasını aç.
3. Build edip çalıştır.
